程小奔的python接口
=====================

本章介绍了程小奔的python接口，一共包含以下几类模块:

* 小程的python接口：主要是指程小奔板载驱动的一些API接口。
* 小奔的python接口：小程用于控制小奔运动或者获取小奔传感器数据的一些API接口。
* 第三方类库接口：程小奔内置的一些第三方类库的接口类，例如 mqtt或者urequest的类。
* 神经元扩展模块的python接口：程小奔扩展神经元模块时的一些API接口。
