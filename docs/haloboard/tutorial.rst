Haloboard's example tutorial
====================

The purpose of this tutorial is to get you started using haloboard.
We need you to have a haloboard, and secondly you need a USB cable (or Bluetooth dongle) to connect to the haloboard.
If you are using Python programming for haloboard for the first time, It is recommended to read this section.
