``四相步进电机`` 模块的主要功能与函数

功能相关函数
----------------------

.. function:: stepping(a,b,c,d,dir,speed)

   步进电机旋转函数，输入参数a,b,c,d分别对应步进电机四相,dir为旋转方向(0或1)，speed为旋转速度最低为2

四相步进电机与开发板接线对应引脚：
------------------------------------

		+------------+-----------+
		| TPYBoard   | 步进电机  |
		+============+===========+
		| VIN        |   ＋      |
		+------------+-----------+
		| GND        |   －      |
		+------------+-----------+
		| X1         |   IN1     |
		+------------+-----------+
		| X2         |   IN2     |
		+------------+-----------+
		| X3         |   IN3     |
		+------------+-----------+
		| X4         |   IN4     |
		+------------+-----------+

程序示例：
----------

.. code-block:: python

  # main.py -- put your code here!
  import pyb
  from stepping motor import stepping

  def main():
	while True:
	stepping('X1','X2','X3','X4',0,2)

  main()
